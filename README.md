# Expense Tracker 🧾

A simple command-line expense tracker built with Python.

## What it does
- Takes your daily budget as input
- Lets you log multiple expenses with categories
- Shows a full summary of all expenses at the end
- Calculates your remaining budget

## What I used
- Python basics: variables, loops, lists, f-strings

## How to run
1. Make sure Python is installed
2. Run: python expense_tracker.py
